# Next.js Todo 연습용 레포 입니다.

## 구현 예정 기능

> 연습용이기때문에 로그인기능은 생략하겠습니다.

- API
  - /todos
    - (GET) : 작성된 todoList를 불러옵니다 ( 페이지 네이션 없이 모든 리스트를 불러옵니다. )
  - /todo
    - (POST) : 새로운 todo를 작성합니다.
  - /todo/[id]
    - (GET) : 해당 id의 todo 상세 정보를 가져옵니다.
    - (PUT) : 해당 id의 todo 를 수정합니다.
    - (DELETE) : 해당 id의 todo를 제거합니다.
